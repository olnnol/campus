# 掌上校园企业微信纯前端复原版 v3

这是一个可直接部署到 GitHub Pages 的纯前端静态项目，复原企业微信内“掌上校园”页面，并实现：

- 首页点击“学生请假申请”进入请假管理列表
- 点击请假卡片进入请假详情
- 请假详情页中的“请假信息”字段可点击修改
- “办理进度”每一行可点击修改节点、状态、人员、时间
- 修改后的数据自动保存到浏览器 localStorage，刷新页面仍保留

## 部署

1. 将本项目所有文件上传到 GitHub 仓库根目录。
2. 进入仓库 `Settings → Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，Folder 选择 `/root`。
5. 等待生成 GitHub Pages 地址。
6. 将生成的地址填入企业微信自建应用的“应用主页”。

## 页面路径

- 首页：`/`
- 请假列表：`#/leave-list`
- 请假详情：`#/leave-detail/20260327`

## 如何修改请假信息

进入请假详情页后，直接点击下面这些行即可修改：

- 请假类型
- 请假去向
- 详细地址
- 开始时间
- 结束时间
- 统计时长
- 紧急联系人
- 紧急联系人电话
- 请假理由
- 请假状态
- 销假状态
- 请假附件

## 如何修改办理进度

进入请假详情页后，点击“办理进度”中的任意一行，可修改：

- 节点名称
- 节点状态
- 办理人
- 办理时间

## 重置数据

如果想恢复初始演示数据，在页面地址后添加：

```text
?reset=1
```

例如：

```text
https://你的用户名.github.io/campus-portal/?reset=1
```

重置完成后再去掉 `?reset=1`。

## 注意

这是纯前端演示版，不包含后端、数据库、企业微信身份认证，也不会真实提交请假申请。
